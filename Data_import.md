``` r
litters_df3 =
```

Use pipes to avoid all this

``` r
litters_df = 
  read_csv("data_import_examples/FAS_litters.csv", na = c("NA", ".", "")) |>
  janitor::clean_names() |>
  select(group, litter_number, starts_with("gd")) |>
  drop_na() |>
  mutate(
    wt_gain = gd18_weight - gd0_weight
  )
```
